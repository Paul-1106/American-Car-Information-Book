With using C++ programming, I am using a CRUD method to modify the list in the "cars.csv" file with using MongoDB as the main database for this project and C++. The "cars.csv" file is added to Mongo Compass to create a database called "carList", one collection which is "cars", and then a document type will be a CSV file.<br><br>

<h2><u>CRUD method:</u></h2>

<h3>✏️ C = Create: </h3><br>
The create function will create a query and then add it to the list in the CSV file.<br><br>

<h3>📖 R = Read: </h3><br>
The read function will search for a query when inputting a specific name of the car and then output the exact car type that user's desire to read the car's information.<br><br>

<h3>♻️U = Update:</h3><br>
The update function will modify a specific query that users will choose if it is either the car's ID, the name, etc. Once editing the query is done, then it will display different information about the car.<br><br>

<h3>🗑️D = Delete:</h3><br>
The delete function will remove a query from the CSV file list or remove the query. The plan for this project is to let users to delete queries by inputting the car's ID.<br><br><br>
